Metallic Crow 
=============

Metallic Crow is a game engine for 2D sprite-based side-scrolling story games.  The engine is written in C++.  It uses SDL for the core multimedia functionality and Box2D for physics.  It is designed to be configured and extended using Lua.  You build your game from Lua scripts which are called from the Metallic Crow executable.
