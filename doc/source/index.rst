Metallic Crow 
=============

Metallic Crow is a game engine for 2D sprite-based side-scrolling story games.  The engine is written in C++.  It uses SDL for the core multimedia functionality, Box2D for physics, Boost for data structures and algorithms and Lua for the embedded extension language.  Metallic Crow is configured and extended using Lua.  You build your game from Lua scripts which are called from the Metallic Crow executable.
